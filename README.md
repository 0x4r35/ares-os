# AresOS - Retro-Futuristic Web Operating System

A browser-based cyberpunk-themed operating system simulation with a stunning retro-futuristic aesthetic.

## Features

### Visual Design
- **Neon Glow** - Cyberpunk color scheme (green, cyan, magenta)
- **CRT Effects** - Authentic scanlines and screen glow
- **Matrix Rain** - Animated background particles
- **Glass Morphism** - Frosted glass window effects
- **Smooth Animations** - 60fps interactions

### Applications
1. **Terminal** - Functional command-line interface
   - Commands: help, date, time, whoami, clear, neofetch, matrix, etc.
   - Cyberpunk-styled output

2. **File Manager** - Visual file browser
   - Folder navigation
   - Grid view with icons

3. **Browser** - Web simulation
   - URL bar
   - Fake loading

4. **Settings** - System configuration
   - Theme colors
   - Sound toggle

5. **Gallery** - Image viewer
   - Cyberpunk-styled textarea
   - Navigate with arrows

6. **Radio** - Music player
   - FM presets
   - Animated visualizer

7. **Code Editor** - Programming environment
   - Syntax highlighting look
   - Line numbers

8. **AI Chat** - NEXUS-AI assistant
   - Interactive responses
   - Easter eggs

## Getting Started

```bash
cd ares_os
# Open index.html in any modern browser
```

Or use a local server:
```bash
npx serve .
```
